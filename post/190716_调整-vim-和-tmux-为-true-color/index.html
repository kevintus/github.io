确认 Gnome-terminal 是支持 True color 的

这一篇已经写的很清楚了： How to enable true-color for neovim, tmux, and gnome-terminal

最终 vimrc 和 tmux.conf 配置

.vimrc 配置

set termguicolors
let &t_8f = "\<Esc>[38;2;%lu;%lu;%lum"
let &t_8b = "\<Esc>[48;2;%lu;%lu;%lum"

set t_Co=256
set background=dark
color molokai
let g:molokai_original=1

.tmux.conf 配置

set -g default-terminal 'screen-256color'
set-option -ga terminal-overrides ",*256col*:Tc"
