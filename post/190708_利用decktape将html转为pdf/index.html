利用 decktape 将 HTML 转为 pdf
利用 decktape，把 reveal-md 的 html 输出导出成为 pdf 文件
2019-07-08 16:30 label reveal-md   decktape  

参考资料

Decktape 官方文档

实现非常直接

npm install -g decktape

# 先用 reveal-md 打开 md 文档
reveal-md xxx.md

# 复制 chrome 里面的 url，再利用 decktape 导出 pdf
decktape reveal <chrome 里面 md 文档 的 url> output.pdf

Done!
