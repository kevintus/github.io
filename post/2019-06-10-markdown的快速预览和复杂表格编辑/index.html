Markdown的快速预览和复杂表格编辑
Sublime Text 3利用 Markdown Preview 插件实现快速预览；同时利用 EXCEL 下的 HTML 格式实现复杂表格编辑
2019-06-10 15:05 label Markdown   Sublime_Text  

Markdown的快速预览

在 Sublime Text 3 里面利用 ctrl + shift + p 呼出菜单，输入 Install 查找到 Markdown Preview 进行安装；之后选择 Preference -> Key Bindings，在 user 配置文件内添加快捷键 Alt + m来实现快速预览，预览文件会在 /tmp 文件夹内

    { "keys": ["alt+m"], "command": "markdown_preview", "args": {"target": "browser", "parser":"markdown"}  }

如需要生成最终HTML文件，快捷键是 Ctrl + b。

利用Excel下的 HTML 格式实现复杂表格编辑
