Markdown的快速预览

在 Sublime Text 3 里面利用 ctrl + shift + p 呼出菜单，输入 Install 查找到 Markdown Preview 进行安装；之后选择 Preference -> Key Bindings，在 user 配置文件内添加快捷键 Alt + m来实现快速预览，预览文件会在 /tmp 文件夹内

    { "keys": ["alt+m"], "command": "markdown_preview", "args": {"target": "browser", "parser":"markdown"}  }

如需要生成最终HTML文件，快捷键是 Ctrl + b。

利用Excel下的 HTML 格式实现复杂表格编辑

原生命令太烦人了，还不支持复杂表格，直接把excel另存为html文件，然后用 gedit 或其他文本编辑软件打开，选取 <table> <\table> 中间部分，粘贴到 md文件里面对应位置即可。 示例如下：

导流渠道 子渠道 主要方式 基础转化率 适用场景
电商（京东、天猫、淘宝） 历史订单 短信、主动添加 xx% 冷启动
销售过程 礼券，页面引导，客服引导，物流信息提醒等 xx% 持续积累
线下导购 门店位置（门口，收银台，POP等），销售服务过程 xx% 持续积累
社交媒体 快手，抖音，小红书，微博 页面引导，内容引导，口播引导等 TBD 持续积累
