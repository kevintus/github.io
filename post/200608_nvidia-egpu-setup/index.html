Setup eGPU in Ubuntu 18.04
2020-06-08 16:11 label eGPU   Nvidia  

参考资料

  • I mainly follow this procedure
  • Adjust the kernel parameter by following this

操作

  1. Purge all existing Nvidia drivers
sudo apt-get --purge remove 'nvidia*'

  1. Reboot sudo reboot

  2. [KEY STEP!!!] Adjust kernel parameters: Edit /etc/default/grub as root, change GRUB_CMDLINE_LINUX line:

sudo vim /etc/default/grub

# In GRUB_CMDLINE_LINUX
GRUB_CMDLINE_LINUX="pcie_ports=native pci=assignbusses,hpbussize=0x33,hpiosize=0,hpmmiosize=256M,hpmmioprefsize=4G,nocrs,realloc iommu=on intel_iommu=on"
  1. Update booting
sudo grub-mkconfig -o /boot/grub/grub.cfg

  1. Reboot

  2. Install Nvidia driver

sudo apt-get install nvidia-driver-440

  1. Reboot

  2. Check the lspci | grep -i VGA and lspci -vnnt and find Nvidia

  3. Update Xconfig

sudo nvidia-xconfig

  1. Reboot

  2. Check with nvidia-smi again, ALL DONE!
