2023.8.8 更新

如果还是用pandoc而不是reveal-md（考虑到reveal-md更适合ppt转pdf，而pandoc更适合word），那么可以生成一个 head.tex 模版文件，然后让pandoc来调用

vim Pandoc-md-to-pdf-head-template.tex

\usepackage{fancyvrb,newverbs}

%% 设置页面间距
\usepackage[top=2cm, bottom=1.5cm, left=2cm, right=2cm]{geometry}

% change background color for inline code in
% markdown files. The following code does not work well for
% long text as the text will exceed the page boundary
\definecolor{bgcolor}{HTML}{E0E0E0}
\let\oldtexttt\texttt

\renewcommand{\texttt}[1]{
  \colorbox{bgcolor}{\oldtexttt{#1}}
}

%% color and other settings for hyperref package
\hypersetup{
    bookmarksopen=true,
    linkcolor=blue,
    filecolor=magenta,
    urlcolor=RoyalBlue,
}

%% 全局中文字体
\setmainfont{Noto Sans CJK SC}

%% 针对中文自动换行
\XeTeXlinebreaklocale "zh"

%% 设置行间距
\linespread{1.2}\selectfont

%% 段落之间的距离
\setlength{\parskip}{3pt}

然后就可以

pandoc –pdf-engine=xelatex -H Pandoc-md-to-pdf-head-template.tex XXX.md -o XXX.pdf

DONE!

环境

  • Ubuntu 内安装 xelatex (pdflatex 不支持中文)
  • 安装中文字体，并用 fc-list :lang=zh 确认

中文显示命令

  • 方式一：命令行输入
pandoc input.md -o output.pdf --latex-engine=xelatex -V mainfont='WenQuanYi Micro Hei'
  • 方式二：在 YAML 输入
---
mainfont: WenQuanYi Micro Hei Mono
---

pdf 页边距调整和自动换行

  • 边距调整

方式一：命令行输入

pandoc input.md -o output.pdf --latex-engine=xelatex -V mainfont='WenQuanYi Micro Hei' -V geometry:margin=1in

方式二：在 YAML 输入

---
geometry:margin=1in
---
  • 自动换行
pandoc -D latex > template.latex

template.latex 模板文件中添加最后一行

$if(mathfont)$
    \setmathfont(Digits,Latin,Greek){$mathfont$}
$endif$
\XeTeXlinebreaklocale "zh"

然后在命令行输入

pandoc --template ~/.pandoc/templates/template.latex

最终 YAML 示例

---

title: Computer Vision Research Notes
author: xxx
date: 2019-07-22

mainfont: WenQuanYi Micro Hei
geometry: margin=1in
fontsize: 12pt
urlcolor: NavyBlue

---

或者在 bashrc 里面设置 Alias 如下：

alias pandoc_md2pdf='pandoc --template ~/.pandoc/templates/template.latex --latex-engine=xelatex -V mainfont="WenQuanYi Micro Hei" -V geometry:margin=1in -V fontsize=12pt -V urlcolor=NavyBlue'

这样只需要输入

pandoc_md2pdf input.md -o output.pdf

Done!
