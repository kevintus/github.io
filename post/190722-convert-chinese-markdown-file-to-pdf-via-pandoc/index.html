环境

  • Ubuntu 内安装 xelatex (pdflatex 不支持中文)
  • 安装中文字体，并用 fc-list :lang=zh 确认

中文显示命令

  • 方式一：命令行输入
pandoc input.md -o output.pdf --latex-engine=xelatex -V mainfont='WenQuanYi Micro Hei'
  • 方式二：在 YAML 输入
---
mainfont: WenQuanYi Micro Hei Mono
---

pdf 页边距调整和自动换行

  • 边距调整

方式一：命令行输入

pandoc input.md -o output.pdf --latex-engine=xelatex -V mainfont='WenQuanYi Micro Hei' -V geometry:margin=1in

方式二：在 YAML 输入

---
geometry:margin=1in
---
  • 自动换行
pandoc -D latex > template.latex

template.latex 模板文件中添加最后一行

$if(mathfont)$
    \setmathfont(Digits,Latin,Greek){$mathfont$}
$endif$
\XeTeXlinebreaklocale "zh"

然后在命令行输入

pandoc --template ~/.pandoc/templates/template.latex

最终 YAML 示例

---

title: Computer Vision Research Notes
author: xxx
date: 2019-07-22

mainfont: WenQuanYi Micro Hei
geometry: margin=1in
fontsize: 12pt
urlcolor: NavyBlue

---

或者在 bashrc 里面设置 Alias 如下：

alias pandoc_md2pdf='pandoc --template ~/.pandoc/templates/template.latex --latex-engine=xelatex -V mainfont="WenQuanYi Micro Hei" -V geometry:margin=1in -V fontsize=12pt -V urlcolor=NavyBlue'

这样只需要输入

pandoc_md2pdf input.md -o output.pdf

Done!
