reveal-md 的格式模版

Posted by     "Kevin L" on Wednesday, June 19, 2019

TOC

基于这个模板，可以比较方便使用 markdown + reveal.js 了。

---
title: xxx
separator: "^\n---\n$" 
verticalSeparator: "^\n--\n$"
theme: black
revealOptions:
    transition: 'slide'

"slideNumber": true
"progress": true
"width": 1920
"height": 1080
"margin": 0.1
"minScale": 0.2
"maxScale": 2

---

<!--Title Page

利用 <h1></h1> 到 <h6></h6> 来控制标题字体大小，正文部分相同 

-->

<h1> PPT 首页主标题</h1>
<h2> 2019.6.18 </h2>

<!--Speaker Note-->
Note: 不会出现在 ppt 里面，会出现在 speaker note 里面

---

<!--Main Page

Some component illustratives:
* 插入一些网页对象 iframe (Prefer to use lazy-loading, ussing `data-src` instead of `src`):

<iframe frameborder="0" width="100%" height="500pt" data-src="https://caniuse.com/cors/embed/description&links"></iframe>

* 插入视频 （youtube，优酷） iframe

<iframe width="100%%" height="500pt" frameborder="0" data-src="http://player.youku.com/embed/XODY3NDMzNjY4" allowfullscreen=""></iframe>

<iframe width="100%" height="500pt" frameborder="0" data-src="https://www.youtube.com/embed/tgbNymZ7vqY"> </iframe>

* 页面为可交互网页

<section data-background-iframe="https://www.gemii.cc" data-background-interactive> <div style="position: absolute; width: 40%; right: 0; box-shadow: 0 1px 4px rgba(0,0,0,0.5), 0 5px 25px rgba(0,0,0,0.2); background-color: rgba(0, 0, 0, 0.9); color: #fff; padding: 20px; font-size: 20px; text-align: left;">
    <h2>Try Website</h2> </section>

* 内容左对齐

<section style="text-align: left;">
    <h2> Thanks & Questions </h2>  
</section>

* 添加脚注：用 sup 添加标注，然后用 footer 进行注释

<sup>[1]</sup>

<footer style="text-align: left; font-size: 0.5em" >
    [1] reference text
</footer>

-->
