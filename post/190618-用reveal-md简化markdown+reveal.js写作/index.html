reveal-md 官方文档

实现

下载 reveal-md

尝试 npm install -g reveal-md 显示失败，安装nvm后，nvm install stable，然后重新 npm install -g reveal-md 提升安装成功

配置文档

研究了一圈文档，有三种实现方式：

  • 命令行，例如
reveal-md slides.md --theme solarized
  • 在 md 文档的目录下新建 reveal.json 和 reveal-md.json 两个配置文档，前者设定文档格式，后者设定每个页面格式，如下:
// reveal.json
{
  "controls": true,
  "progress": true
}

// reveal-md.json
{
  "separator": "^\n\n\n",
  "verticalSeparator": "^\n\n"
}
  • 利用 YAML Front matter 在文档内进行配置，如下：
---
title: Foobar
separator: <!--s-->
verticalSeparator: <!--v-->
theme: solarized
revealOptions:
    transition: 'fade'
---
Foo

Note: test note

<!--s-->

# Bar

<!--v-->

最后选择了第三种方式，因为第二种还得每次把 md 文档放在带有那两个.json的文件夹内

我的一个 md 文档格式示例

---
title: xxx
separator: "^\n---\n$"
verticalSeparator: "^\n--\n$"
theme: black 
revealOptions:
    transition: 'slide'

"slideNumber": true
"progress": true
"width": 1920
"height": 1080
"margin": 0.1
"minScale": 0.2
"maxScale": 2

---

这样可以把文档放在任何位置，然后每次用 reveal-md XX.md 输出最终 html。
